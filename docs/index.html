How it works

Three steps to a clean library

No complex settings. No learning curve. Dedupix gets out of the way and lets you focus on keeping the photos that matter.

1

Select your folder

Drop a folder or browse to it. Dedupix supports all major image formats and scans sub-folders automatically.

2

Let it scan

Dedupix uses SHA-256 hashing to find exact duplicates — pixel-perfect matches, not just file names. Scans thousands of files in seconds.

3

Review & delete

See every duplicate group side-by-side. Choose which copies to keep, then delete in one click. Your originals stay safe.

Features

Built for people with real photo libraries

Every feature is designed around how people actually use their photo collections.

SHA-256 hashing

Byte-perfect duplicate detection. Two files with different names but identical content are always caught.

Recursive scanning

Scan entire folder trees at once. Dedupix digs into nested sub-folders automatically so nothing is missed.

10+ image formats

JPG, PNG, GIF, BMP, TIFF, WEBP, HEIC, RAW and more. Filter by extension or scan everything at once.

Visual review

See thumbnails side-by-side before deleting anything. Compare file size and date to make smart decisions.

100% local

Your photos never leave your machine. Dedupix runs entirely offline — no cloud, no telemetry, no account.

Fast as lightning

Scans thousands of photos in seconds. A 10,000-file library typically completes in under a minute.

Smart scanning

Finds duplicates you'd never spot manually

Renamed files, copies in different folders, exports with slightly different names — Dedupix catches them all using content-based hashing, not just file names.

  • Exact pixel-perfect match detection
  • Works across nested folder structures
  • Handles renamed and moved files
  • Shows file size, date, and full path for each copy

Safe by design

You decide what gets deleted. Always.

Dedupix never auto-deletes. Every action requires your confirmation. Review the full duplicate group, pick the keeper, then delete the rest — one group at a time or all at once.

  • Confirm before any deletion
  • See original vs duplicate side by side
  • Auto-selects the best copy to keep
  • Delete all at once or group by group
